From Galacticraft Wiki
Jump to: navigation, search
[edit] [history] [purge] Template documentation

This template warn the reader that the article is a disambiguation page. It should be used at the bottom of all disambiguation pages.

Pages with this template on are added to Category:Disambiguation pages.

This also allows those pages to be found on Special:Disambiguations.

Use

  • {{Disambig}}
This disambiguation page lists articles associated with the same title. If an internal link led you here, you may wish to change the link to point directly to the intended article.


  • {{Disambig|nocat}}

Used to prevent the template adding the page to Category:Disambiguation pages.
