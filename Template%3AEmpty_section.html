Usage

Common form

{{Empty section|date=April 2023|section=}}

  • {{Empty section}} puts articles into Category:Articles to be expanded.

Parameters

date

If a date parameter is provided, as in the example above, then the page is put into a by-month subcategory instead. If the date is left out, a bot will add it later.

section
(optional) The section number to be edited (or whatever the &action=edit&section= parameter accepts).
small
Anything but 'left' ' = Uses an in-line default-sized message box.
left (default) = Uses smaller left-aligned message box designed for the head of sections. (See Template:Ambox/doc#The small parameters )
